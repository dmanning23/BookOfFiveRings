### How to play

Put the slicey part of the katana into the opponent. 

First person to die, loses.

### Controls

| Action        | Result  |
|:------------- | -----:|
| Left      | Move left |
| Right      | Move right |
| Button | Attack |
| Up (immediately after attacking)| Cancel attack |
| Down | Block |
| Yell | KIAI! |
| Meditate | Zanshin |
| Unplug other controller | Disarm opponent |
